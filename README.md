# poo-pbl-grupo-mosqueteiros
este programa é a avaliação final para a disciplna projeto de programação, do terceiro período de Ciências da Computação na UNIMA 
dado os temas propostos pelo professor, foi utilizado o tema 3. Sistema de Logística e Entregas Urbanas 

## como rodar o projeto
Abra o projeto em uma IDE para acessar-lo, execute o arquivo main dentro da pasta scr/main/java/presentation
utilize-se do console para criar pedidos, frotas, roteiros e intinerarios, cada objeto é armazenado em um arraylist especifico para seu tipo
foram implementados modelos de arquitetura para a programação orientada a objetos, polimorfismo, e testes para Test-Driven Development

## pastas do codigo fonte
a pasta domain contem os varios objetos utilizados pelo programa
a pasta application contém a classe cadastrador, responsável por gerenciar os vários métodos para a interação com os objetos
a pasta presentation contém o main, onde o usuário irá inserir e receber informações
a pasta test contém os vários testes para java, necessários para checar que os objetos estão sendo criados corretamente
